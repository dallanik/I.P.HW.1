Список файлов в данном репозитории и их назначение.

1)Timechanger.py - модуль, реализующий функцию, осуществляющюю изменение времени на указанную константу секунд.
2)Server.py - Сервер, после включения на каждое установленное соединение высылает "неправильное" время.
3)Client.py - Клиент, устанавливает связь с сервером, считывает данные и выводит их в консоль.
4)config.txt - файл, содержащий константу, на которую происходит смещение времени.
5)Test.txt - Результаты нескольких пробных запусков с различными входными данными.

Справка по использованию:

1)Установите необходимое значение константы в файле config.txt. Значение должно быть целым числом.
2)Запустите Server.py для "поднятия" сервера.
3)Запустите Client.py. Он установит соединение с поднятым нами сервером, считает оотуда данные и выведет на консоль.
